# PDF Split and Merge Tool

This Python script helps you split a large PDF into smaller parts or merge many PDFs into one file.

It is useful when you need to send only a few pages or combine multiple documents for clients.

## What it can do

- Split a PDF into selected page ranges
- Merge multiple PDF files into a single file
- Keep the original files unchanged

## How to use

1. Put your PDF files in the same folder as this script.  
2. Open the script and set the input file names and page ranges.  
3. Run the script with Python.  
4. New PDF files will be created for split or merged output.
