## What this folder contains

This folder has tools for working with PDF documents.  
They are useful for invoices, reports, e‑books and scanned files.

- `pdf_split_merge.py` – splits big PDFs or merges many PDFs into one  
  You can select page ranges to split a file, or combine several small PDFs into a single document.

- `pdf_rename.py` – renames many PDF files using a simple pattern  
  Helps you clean names like "scan001.pdf, scan002.pdf" into clear names based on date, client or project.
